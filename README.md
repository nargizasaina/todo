

## Project setup

```bash
$ npm install
```

## Migration

```bash
$ npx prisma migrate dev
$ npx prisma generate
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Run tests

```bash

# e2e tests
$ npm run test:e2e
```
